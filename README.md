# ☁️ CloudBreach Range

> Learn Cloud Security by Building, Breaking, and Securing AWS Environments.

CloudBreach Range is a hands-on cloud security laboratory where intentionally vulnerable AWS environments are created, exploited, analyzed, and secured.

The objective is not only to understand cloud attacks but also to engineer secure cloud architectures using Infrastructure as Code (Terraform), cloud-native security services, and security best practices.

This project is part of my journey toward becoming a Cloud Security Engineer with expertise in offensive security, defensive engineering, and AI-assisted security automation.

---

## 🎯 Objectives

- Learn AWS Cloud Security through practical labs
- Understand real-world cloud attack paths
- Practice Infrastructure as Code using Terraform
- Develop an attacker mindset
- Learn defensive cloud engineering
- Build a professional cloud security portfolio
